Install IntelliJ
  1. 학생 라이센스 발급 방법
  2. 학생 라이센스 신청 링크에 접속합니다.
  3. Apply now 버튼을 클릭 후 대학 이메일 주소 방식으로 신청합니다.
  4. 제주대학교 이메일은 제주대학교 포털사이트에 접속해서 확인 할 수 있습니다.
    제주대학교 포털시스템
  5. 제주대학교 포털 사이트 메일에 들어가 인증 메일을 확인 후 Confirm Request 클릭
  6. "JetBrains Account" 사이트 창이 열리고, "I Accept"을 클릭
  7. 로그인하여 학생 라이센스가 발급 된 것을 확인하면 학생 라이센스는 발급 완료입니다.
  • 학생 라이센스를 발급 완료 했다면 이제 인텔리제이를 설치해보겠습니다.
  • JetBrains사이트에 접속하고 Download버튼을 클릭한다
  • 저희는 학생 라이센스를 발급 받았기때문에 Ultimate로 다운로드 해줍니다.
    • 이미지
  • 인텔리제이 설치가 완료되면, 인텔리제이를 실행 하여 최초 인증 계정 정보를 입력하면 사용 가능하다
